
## 简介

- 设备出厂前的测试工具

## 连接 

- 自动连接   

   - 打开工具自动扫描：Timeout - 3s  

   - 蓝牙信号强度 (Rssi) 超过预设值：停止扫描，连接  

   - 断开连接 (设备关机) 后：自动扫描
  
- 连接参数
 
  - Riss  

  - Devices Name  
  
## 功能  

- 图形化界面  

  - 键盘按键  

      图标动态变化
      
  - 鼠标  

      鼠标指针图标 + 轨迹

## 菜单  

- Rssi 调节  

  - 拖动条  

      调节 Rssi 门槛
